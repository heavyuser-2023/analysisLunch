인스타그램 운영자가
준비해 주실 두 가지

점심봇이 사장님 인스타그램에 자동으로 글을 올리려면, “올려도 된다”는 허락이 담긴 값 두 개가 필요해요. 개발 지식 없이 화면에서 클릭만으로 받을 수 있게 정리했습니다.

1

게시 허락 열쇠

점심봇이 내 인스타에 글을 올릴 수 있게 해주는 비밀 열쇠.

INSTAGRAM_ACCESS_TOKEN
2

내 계정 번호

글을 올릴 “바로 그 인스타 계정”을 가리키는 숫자.

INSTAGRAM_BUSINESS_ACCOUNT_ID
⏱️ 처음 한 번만 하면 끝이에요. 천천히 해도 20~30분 정도. 중간에 막히면 담당 개발자에게 화면을 공유하며 같이 진행하셔도 됩니다.
1

인스타 계정 준비

스마트폰 · 약 5분

자동 게시는 “비즈니스(프로페셔널)” 계정에서만 됩니다. 아래 3가지를 먼저 맞춰주세요. 이미 돼 있으면 2단계로 넘어가셔도 됩니다.

  1. 인스타를 “프로페셔널 계정”으로 바꾸기

    개인 계정으로는 자동 게시가 안 돼요. 무료이고, 팔로워에게 보이는 건 달라지지 않습니다.

    • 인스타 앱 → 내 프로필 → 우측 상단 ☰ → 설정 및 개인정보
    • 계정 유형 및 도구 → 프로페셔널 계정으로 전환
    • 비즈니스 선택

  2. 페이스북 페이지 만들기

    인스타 자동 게시는 규정상 반드시 페이스북 페이지를 거쳐야 해요. (개인 페이스북 말고 “페이지”입니다. 운영하지 않아도 만들기만 하면 됩니다.)

    • 브라우저에서 facebook.com/pages/create 접속
    • 페이지 이름(예: 가게 이름)만 적고 만들기 → 끝

  3. 인스타와 페이스북 페이지 연결

    • 인스타 앱 → 프로필 → 프로필 편집
    • 페이지 항목에서 방금 만든 페이스북 페이지 선택
    여기까지 완료되면 내 인스타 계정과 페이스북 페이지가 한 쌍으로 묶였어요. 이게 모든 것의 토대입니다. 🎉
2

개발자 사이트 입장 & 앱 만들기

컴퓨터 권장 · 약 10분

이름이 “개발자 사이트”라 겁나지만, 우리는 버튼만 몇 번 누릅니다. 이 사이트에서 점심봇이 인스타에 접근할 “통로(앱)”를 하나 만들어요.

  1. Meta 개발자 사이트 로그인

    • 컴퓨터 브라우저에서 developers.facebook.com 접속
    • 위에서 만든(또는 사용 중인) 페이스북 계정으로 로그인

  2. 앱 만들기

    • 우측 상단 내 앱 → 앱 만들기
    • 유형을 묻는 화면에서 “비즈니스(Business)” 선택 → 다음
    • 앱 이름 입력 (예: 점심봇 연동) → 만들기

  3. 인스타그램 기능 추가

    • 앱 화면 왼쪽/가운데에서 제품 추가 목록을 찾기
    • Instagram Graph API(또는 “Instagram”) 옆 설정/추가 클릭
    막히면 화면 구성은 Meta가 가끔 바꿔요. 단어가 조금 달라도 “Instagram”이 들어간 항목을 추가하면 됩니다. 헷갈리면 이 화면을 캡처해 개발자에게 보내주세요.
3

① 게시 허락 열쇠 받기

→ 첫 번째 값

이제 점심봇이 쓸 “열쇠”를 만듭니다. 일반 열쇠는 60일이면 끊기니, 우리는 끊기지 않는 열쇠를 만들 거예요.

  1. 비즈니스 설정 들어가기

    • 브라우저에서 business.facebook.com/settings 접속
    • 왼쪽 메뉴에서 사용자 → 시스템 사용자
    시스템 사용자란? 사람이 아니라 “자동화 전용 직원”이라고 생각하면 돼요. 이 직원의 열쇠는 비밀번호 바뀜 같은 일과 무관해 잘 안 끊깁니다.

  2. 자동화용 사용자 추가

    • 추가 → 이름 입력 (예: lunch-bot)
    • 역할은 관리자 선택 → 만들기

  3. 이 사용자에게 권한 주기

    방금 만든 사용자를 클릭한 뒤 자산 할당:

    • 페이지 선택 → 1단계에서 만든 페이스북 페이지 고르기 → “콘텐츠 관리/전체 제어” 켜기
    • 앱도 같은 방식으로 2단계에서 만든 앱 할당

  4. 열쇠(토큰) 만들기 — 가장 중요한 단계

    같은 화면에서 토큰 생성 클릭:

    • 앱: 2단계에서 만든 앱 선택
    • 토큰 만료: “안 함(Never)” ← 꼭 이걸로! (끊기지 않게)
    • 권한 체크박스에서 아래 5개를 켜기: instagram_basic instagram_content_publish pages_show_list pages_read_engagement business_management
    • 생성 → 길게 나온 문자열을 통째로 복사
    딱 한 번만 보여요 이 열쇠는 만든 직후에만 전부 보입니다. 지금 바로 복사해서 메모장 등 안전한 곳에 붙여넣으세요. 놓치면 처음부터 다시 만들어야 해요.
    받았어요 ✓ 복사한 이 긴 문자열이 첫 번째 값(게시 허락 열쇠)입니다.
4

② 내 계정 번호 받기

→ 두 번째 값

마지막으로 “어느 인스타에 올릴지”를 가리키는 숫자를 받아요. 터미널 없이 Meta가 제공하는 조회 화면에서 클릭으로 확인합니다.

  1. 조회 화면 열기

    • 브라우저에서 developers.facebook.com/tools/explorer 접속
    • 우측 상단 Meta App 칸에서 2단계에서 만든 앱 선택

  2. 내 페이지 찾기

    가운데 주소 입력칸에 아래를 그대로 적고 Submit(제출):

    • 입력칸에 me/accounts 입력 → 제출
    • 아래에 페이지 목록이 나와요. 내 페이지 이름 옆 "id" 숫자를 복사 (이건 임시로 쓰는 페이지 번호)

  3. 인스타 계정 번호 꺼내기

    입력칸 내용을 아래처럼 바꿔 다시 제출 (붉은 부분에 방금 복사한 페이지 번호를 넣으세요):

    • 입력: 페이지번호?fields=instagram_business_account
    • 결과에서 instagram_business_account 안의 id 숫자(보통 17841…로 시작)를 복사
    받았어요 ✓ 그 17841…로 시작하는 숫자가 두 번째 값(내 계정 번호)입니다.
    이 단계가 가장 헷갈려요 조회 화면이 낯설면, 여기까지 오신 김에 개발자에게 화면 공유로 같이 두 줄만 조회하는 걸 추천합니다. 1분이면 끝나요.
✓

두 값 전달하기

개발자에게

아래 두 칸을 채워 담당 개발자에게 안전한 방법(메신저 DM 등)으로 전달하면 끝입니다. 이후 자동 게시는 점심봇이 알아서 합니다.

📋 전달 양식

① 게시 허락 열쇠 (아주 긴 문자열)
EAAG… 로 시작하는 긴 값
② 내 계정 번호 (숫자)
17841… 로 시작하는 숫자
보안 주의 ①번 열쇠는 비밀번호와 같아요. 공개된 곳(오픈 채팅, 게시판, 메일 제목)에 절대 올리지 마세요. 혹시 노출됐다면 3단계를 다시 해 새 열쇠를 만들면 이전 것은 무효가 됩니다.
나중에 갑자기 안 올라간다면 열쇠를 만든 계정의 비밀번호 변경, 권한 회수, 자동화 사용자 삭제 등이 있으면 열쇠가 끊길 수 있어요. 그럴 땐 3단계만 다시 해서 새 열쇠를 전달해 주시면 됩니다.
